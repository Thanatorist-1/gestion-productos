# GestionaYa — Proyecto React

## Requisitos
- Node.js instalado
- Git

## Cómo arrancar

1. Clonar el repo
   git clone https://github.com/Thanatorist-1/gestion-productos.git

2. Entrar a la carpeta
   cd gestion-productos

3. Instalar dependencias
   npm install

4. Correr el proyecto
   npm run dev

## Reglas del equipo
- Solo Tailwind para estilos, nada de Bootstrap
- Cada quien trabaja en su archivo de página asignado
- Componentes reutilizables van en src/components/ui/
- Hacer git pull antes de cada git push
